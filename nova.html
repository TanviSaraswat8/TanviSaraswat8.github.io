01 · About

What Nova does.

Nova is a voice-activated desktop assistant built in Python. You speak — it listens through your laptop microphone, processes what you said, and responds or acts immediately.

No wake word services, no cloud dependency for core functions. Just Python running locally on your machine, responding to your voice in real time.

"open youtube"
Opens YouTube directly in your browser
"what time is it"
Speaks back the current time
"play [song name]"
Searches and plays on YouTube
"open google"
Launches Google in browser
"what's the date"
Reads out today's date
"open spotify"
Opens Spotify web player
02 · Stack

Built with Python.

Nova uses four core Python libraries, each handling a specific layer of the voice interaction pipeline.

speech_recognition Captures audio from the microphone and converts speech to text using Google's API Voice Input
pyttsx3 Text-to-speech engine — makes Nova speak responses out loud, offline Voice Output
webbrowser Opens URLs and web services on voice command Automation
datetime / os Fetches system time, date, and executes system-level commands System
03 · Code

How it works.

The core listen-respond loop — Nova captures audio, transcribes it, matches a command, and acts.

def listen(): with sr.Microphone() as mic: audio = r.listen(mic, phrase_time_limit=5) try: command = r.recognize_google(audio) return command.lower() except: speak("Please say that again.") return "" # Main command loop elif "open youtube" in command: speak("Opening YouTube") webbrowser.open("https://youtube.com")
